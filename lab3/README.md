### Шаг 1. Создание массива транзакций

Создала файл `main.js` и добавила в него массив объектов с транзакциями. Каждый объект содержит 7 свойств: `transaction_id`, `transaction_date`, `transaction_amount`, `transaction_type`, `transaction_description`, `merchant_name`, `card_type`.

```javascript
    transaction_amount: 100.0,
    transaction_type: "debit",
    transaction_description: "Payment for groceries",
    merchant_name: "SuperMart",
    card_type: "Visa",
  },
];
```
---
### Шаг 2. Реализация функций

Реализовала все требуемые функции. Для функции `calculateTotalAmountByDate` пришлось использовать `new Date()` для разбора строки с датой и сравнивать год, месяц и день по отдельности. Также использовала объект для подсчёта транзакций по месяцам в `findMostTransactionsMonth`:

```javascript
function findMostTransactionsMonth(transactions) {
  let monthCount = {};
  for (let i = 0; i < transactions.length; i++) {
    let month = new Date(transactions[i].transaction_date).getMonth() + 1;
    if (monthCount[month] === undefined) {
      monthCount[month] = 0;
    }
    monthCount[month]++;
  }
}
```

Для получения уникальных типов транзакций использовала `Set`:

```javascript
function getUniqueTransactionTypes(transactions) {
  let types = new Set();
  for (let i = 0; i < transactions.length; i++) {
    types.add(transactions[i].transaction_type);
  }
  return Array.from(types);
}
```

## Контрольные вопросы

### Какие методы массивов можно использовать для обработки объектов в JavaScript?

 Можно использовать разные методы массивов, например, метод `forEach()` подходит для простого перебора элементов, когда нужно выполнить действие для каждого объекта. Метод `filter()` используется для отбора элементов по условию, например для получения только транзакций типа `"debit"`. Метод `find()` позволяет найти первый объект, который подходит под условие, например транзакцию по определённому id.

Также можно использовать метод `map()`, если нужно создать новый массив на основе старого, и метод `reduce()`, если нужно получить одно итоговое значение, например сумму всех транзакций.

---

### Как сравнивать даты в строковом формате в JavaScript?

Если даты записаны в формате `"YYYY-MM-DD"`, их можно сравнивать как строки, потому что сначала идёт год, потом месяц и день. При таком формате порядок строк совпадает с порядком дат.

Но более надёжный способ является преобразовать строку в объект `Date` с помощью `new Date()`. После этого даты можно сравнивать операторами `<`, `>`, `<=`, `>=`.

---

### В чём разница между map(), filter() и reduce() при работе с массивами объектов?

Метод `map()` создаёт новый массив той же длины, но преобразует каждый элемент. Например, из массива транзакций можно получить массив только с их описаниями.

Метод `filter()` создаёт новый массив только из тех элементов, которые подходят под условие. Например, можно оставить только транзакции типа `"debit"` или транзакции с суммой в нужном диапазоне.

Метод `reduce()` используется, когда нужно получить одно итоговое значение из всего массива. Например, с его помощью можно посчитать общую сумму всех транзакций.

То есть `map()` преобразует элементы, `filter()` отбирает нужные элементы, а `reduce()` сводит массив к одному результату.
